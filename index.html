Backend Development

Creation and maintenance of the server-side components that power web or mobile applications. It focuses on building the logic and functionality behind the scenes, enabling communication between the frontend (user interface) and the server, and handling data processing and storage.

Frontend Development

Creation and maintenance the user-facing components of a web application. It involves writing code that defines the visual and interactive elements that users see and interact with directly. I can work closely with designers and backend developers to bring the application's user interface (UI) and user experience (UX) to life.
